# ToDO App

A cross-platform **To-Do application** built with **Flutter, Dart, and Firebase** to help users manage daily tasks efficiently.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Real-time task storage using **Firebase**
- User-friendly and responsive interface
- Works on both Android and iOS

## Technologies Used
- **Frontend & Backend:** Flutter & Dart
- **Database & Authentication:** Firebase
